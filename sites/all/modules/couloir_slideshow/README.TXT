Description
-----------

Couloir Slideshow is a module which displays a javascript slideshow from a folder of images as a block.
Based on the original Couloir slideshow found at http://www.couloir.org/js_slideshow/


Requirements
------------

This module works with Drupal 6.x
In order for Lightbox effects Lightbox2 must be installed


Installation
------------

1) Copy/upload the contents of the couloir_slideshow folder into a
   subdirectory called couloir_slideshow/ in the sites/all/modules/ directory of
   your Drupal installation.
2) Enable the Couloir Slideshow module (Administer -> Site building -> Modules).
3) Configure at Administer -> Site configuration -> Couloir slideshow.


Configuration
-------------

Couloir Slideshow settings can be configured at
  Administer -> Site configuration -> Couloir slideshow.

Configure each block individualy on the block configuration page.
